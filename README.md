## funnyCoding
coding for fun, coding for everything

## 初衷
很多时候编程只是为了实现某个小功能，并不是想做一个完整的系统。但是这样的小功能往往需要基本的业务组件支持，以便更好的展示。
有时一些有趣的idea转瞬即逝，以后尽可能的在 funnyCoding 中记录下来。
funnyCoding 并不是一个“后台管理系统”，它的每一个模块相互独立，互不影响。
